2208.将数组和减半的最少操作次数

发表于 2022-03-19 更新于 2022-10-30 分类于 LeetCode刷题

2208.将数组和减半的最少操作次数

题目描述

一个正整数组,每次操作可以选择一个数将他减小到一半,减小到一半的数依然在原数组可以继续操作，求最少操作多少次后可以使原数组和减半

思路

先排序,然后求和,然后每次都取最大的数进行减半,判断已减的和是否到达了原数组和的一半

超时的原因及优化

减半后的数我插入到了原数组中进行插入排序O(n^2),所以超时了。

设置一个额外的数组保存已减半的数，令index2指向该数组中还未使用过的最大的数，length是该数组的长度

每次比较原数组的当前最大数和index2指向的数，然后将减半的数放到这个数组的后面

可以证明这个数组每次向尾部增加的元素总是小于前一个元素

这样就通过O(n)的时间解决了，成功ac

386.字典序排数

题目描述

将0~n范围内的数按字典序排列

如1,10,11,12,13,2,3,4,5,6,7,8,9

解题思路

1.O(nlogn):按序生成1-n的list然后排序，然后返回

2.建立一个字典树，然后前序遍历

但是这样会使用O(n)的额外空间，所以在建树的过程中直接保存当前值即可，不保留树，这样就是题目要求的O(1)额外空间。

6021.字符串中最多数目的子字符串

发表于 2022-03-19 更新于 2022-09-18 分类于 LeetCode刷题

6021.字符串中最多数目的子字符串

题目描述

给一个字符串text和一个字符串pattern,pattern的长度是2,在text的任意位置插入pattern[0]或pattern[1],求插入后text中最多包含多少个等于pattern的子序列

子序列指的是将一个字符串删除若干个字符后（也可以不删除），剩余字符保持原本顺序得到的字符串。

解题思路

将pattern[0]插入到text的头部或将pattern[1]插入到pattern的尾部,然后进行判断即可。

O(n)的解法,使用cnt记录pattern[0]出现的次数，当pattern[1]出现时，ans+=cnt;

我超时的原因

先遍历一遍text将其中的pattern[0]或pattern[1]字符拿出来

然后在新的字符串中找pattern[0],然后再从该patter[0]的索引位置向后寻找patter[1]的数量,相当于o(n^2)的时间复杂度，所以超时了。

思考:逆向思维，找pattern[1]同时记录之前出现过的pattern[0]的个数即可。

设置一个集合保存有用的单词

遍历words，将长度为1的或者word的子串(0,length-1)在集合中，就将该word加入集合

同时更新ans
