942.增减字符串匹配

发表于 2022-05-09 分类于 LeetCode刷题

942.增减字符串匹配

题目描述

由范围[0,n]内所有整数组成的n+1个整数的排列序列可以表示为长度为n的字符串s，其中:

如果 perm[i] < perm[i + 1] ，那么 s[i] == 'I' 
如果 perm[i] > perm[i + 1] ，那么 s[i] == 'D' 

给定一个字符串s，重构排列perm并返回它。如果有多个有效排列perm，则返回其中任何一个。

解题思路

贪心：

min=0，max=n

s[i]=='I'时，就直接令perm[i]=min，同时min++

s[i]=='D'时，就直接令perm[i]=max，同时max--

最后给perm[n]赋值，并返回perm

442.数组中重复的数据

题目描述

一个长度为n的整数数组nums，其中nums的所有整数都在范围[1,n]内，且每个整数出现一次或两次

找出所有出现两次的整数，并以数组形式返回

要求：时间复杂度为 O(n) 且仅使用常量额外空间的算法解决此问题。

解题思路

因为长度为n，且数据范围是[1,n]，用桶的思想，将nums[i]放到nums[i]-1的下标处，其他数字无法重复放置，这样就可以判断出是否重复了。

看题解发现这个思路叫原地hash：hash(value)=value−1

433.最小基因变化

题目描述

基因序列可以表示为一条由8个字符组成的字符串，其中每个字符都是'A'、'C'、'G'和'T'之一

假设我们需要调查从基因序列start变为end所发生的基因变化。一次基因变化就意味着这个基因序列中的一个字符发生了变化

例如，"AACCGGTT" --> "AACCGGTA"就是一次基因变化

另有一个基因库bank记录了所有有效的基因变化，只有基因库中的基因才是有效的基因序列

给你两个基因序列start和end，以及一个基因库bank，找出并返回能够使start变化为end所需的最少变化次数

如果无法完成此基因变化，返回-1

注意：起始基因序列 start 默认是有效的，但是它并不一定会出现在基因库中。

解题思路

如果start不在bank里，将其放入bank，如果end不在bank里，返回-1

根据bank的内容构建图，如果两个染色体的基因相差位数为1，这两个染色体就存在一条边

从start开始BFS，直到找到end，如果找不到返回-1。找到了就返回路径
