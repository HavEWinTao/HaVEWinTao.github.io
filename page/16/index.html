198.打家劫舍

题目描述

给定一个代表每个房屋存放金额的非负整数数组，计算你 不触动警报装置的情况下，一夜之内能够偷窃到的最高金额。 如果两间相邻的房屋在同一晚上被小偷闯入，系统会自动报警。

解题思路

dp[i][0]表示不拿第i个屋，dp[i][1]表示拿第i个屋

不偷i屋，那么dp[i][0]应该是偷或者不偷i-1屋的最大值

偷i屋，那么dp[i][1]应该是不偷i-1屋的值+该屋的值

142.环形链表II

题目描述

给定一个链表的头节点head，返回链表开始入环的第一个节点。如果链表无环，则返回null。

解题思路

141.环形链表是判断链表是否存在环，该题是找到环的入口

如果不存在环，那么快指针一定会走到null

如果存在环，设链表的左侧非环长度是m，环形长度是c

快指针走了2n个结点，慢指针走了n个结点，那么慢指针就在环内走了n-m个结点

让慢指针再在环内走m个结点那么他就来到了环的起始位置，这样的话可以另取一指针指向头部，这个指针走过m个结点时正好和慢指针在环的起始位置相遇，作为答案返回即可

1606.找到处理最多请求的服务器

题目描述

k个服务器，一个服务器同一时间只能处理一个请求

一个arrive[]请求到达时间数组，一个load[]请求工作时间数组

第i个请求优先分配给第i%k个服务器，如果该服务器繁忙则继续查找下一个服务器，这些服务器视作一个环

返回最繁忙的服务器列表

解题思路

用一个数组记录每个服务器的执行任务的数量

用一个TreeSet来记录可用的服务器isAvailable

用一个优先队列isBusy记录任务完成时间以及这个服务器序号，按完成时间升序排序

开始模拟任务达到的过程

1. 记录到达时间
2. 如果有已经执行完任务的服务器将他从isBusy中弹出并且加入到isAvailable中去
3. 如果isAvailable为空这个任务执行不了
4. 否则按规则找到可以执行的服务器
5. 更新isAvailable和isBusy

最后遍历每个服务器执行任务的数量并得出答案

236.二叉树的最近公共祖先

题目描述

给定一个二叉树, 找到该树中两个指定节点的最近公共祖先

一个节点也可以是它自己的祖先

解题思路

这个题大二小学期好像碰到过类似的，当时感觉老难了，现在竟然一下写出来了

用一个Map记录每个节点的父节点，一遍DFS完成Map的构建

然后记录一个结点的父节点的路径，用Set记录即可

然后查找另一个结点的父节点路径，并判断是否出现在Set中，第一次出现在Set中的就是最近公共祖先

2024.考试的最大困扰度

发表于 2022-03-29 更新于 2022-03-30 分类于 LeetCode刷题

2024.考试的最大困扰度

题目描述

一个由T和F组成的字符串，最大化连续相同的字符，可以将一个字符修改为T或F，最多修改k次

解题思路

两个指针left和right记录[left right]内T和F的数量

left和right都等于0，将right右移遍历整个字符串

如果T,F数量中的最小值比k大了，说明以及无法修改了，将left左移，然后修改移动后的[left right]的数量，直到满足T,F数量中的最小值等于k

然后每次循环中都维护[left,right]的最大长度
