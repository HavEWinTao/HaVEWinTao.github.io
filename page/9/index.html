238.除自身以外数组的乘积

题目描述

给一个整数数组nums，返回数组answer，其中answer[i]等于nums中除nums[i]之外其余各元素的乘积

要求：O(n)并且不能使用除法，保证数据在INT范围内

解题思路

将answer[i]分解为L[i-1],R[i+1]

其中L[i-1]是从左向右乘，从第0个元素乘到第i-1个元素，R[i+1]是从右往左乘，从最后一个元素乘到第i+1个元素

再处理一下answer[0]和answer[n-1]的边界即可

215.数组中的第K个最大元素

题目描述

给定整数数组nums和整数k，返回数组中第k个最大的元素

需要找的是数组排序后的第k个最大的元素，而不是第k个不同的元素

也就是数组排序后的nums[k]

解题思路

  1. 小顶堆

    使用一个容量为k的小顶堆，小顶堆中保存的是k个最大的元素，当当前数比堆顶元素大的时候，需要更新小顶堆

    最后返回堆顶元素即可

    这样只保留k个最大的，空间较优，数据不需要一次性读入内存

    时间复杂度：O(NlogK)

  2. 基于快速排序的快速选择

    对于q,左区间都比a[q]小，右区间都比a[q]大

    当q是要找的第k大时候直接返回nums[q]，不然根据条件再次对左右区间进行操作

通过这道题对快速排序的理解加深了

题目描述

现有一种使用英语字母的外星文语言，这门语言的字母顺序与英语顺序不同。

给定一个字符串列表words，作为这门语言的词典，words 中的字符串已经按这门新语言的字母顺序进行了排序

根据该词典还原出此语言中已知的字母顺序，并按字母递增顺序排列。若不存在合法字母顺序，返回""。若存在多种可能的合法字母顺序，返回其中任意一种顺序即可

1. 在第一个不同字母处，如果s中的字母在这门外星语言的字母顺序中位于t中字母之前，那么s的字典顺序小于t。

2. 如果前面min(s.length, t.length)字母都相同，那么s.length<t.length时，s的字典顺序也小于t

148.排序链表

题目描述

给定一个链表，按升序排序，并返回头结点

解题思路

归并排序,写了好久发现又写的有点墨迹，我想直接对node进行连接这样可以省一些空间，然后按照数组归并的思路head,mid,tail分成区间，然后将每个区间排序

这样就导致了一些bug，比如(mid,tail)区间中的顺序变化了，导致了合并时无法正确判断(head,mid)区间的中止，导致错误

题解将数组划分区间时使这个区间的下一个结点为null

所以在合并时也很方便，直接判断是否为null就可以判断该区间是否结束了，也不会存在区间结束的判断问题

780.到达终点

题目描述

给定四个整数sx,sy,tx和ty

如果通过一系列的转换可以从起点(sx,sy)到达终点(tx,ty)，则返回true，否则返回false

从点(x,y)可以转换到(x,x+y)或者(x+y,y)。

解题思路

正向计算情况很多会超时，所以逆向回推，由tx,ty判断能不能回到sx,sy

所以逆推的话就是拿大数减去小数

//tx和ty有一个大小关系，两个数都取对方的余数，就只对较大数生效

return reachingPoints(sx, sy, tx % ty, ty % tx);

当有一个数已经和其实点坐标相同时，只能动另一个数了，判断(ty-sy)%sx==0或者(tx-sx)%sy==0

当tx,ty之一已经比sx,sy小的时候，就返回false

429.N 叉树的层序遍历

题目描述

层次遍历，返回每层的元素

解题思路

就是常规的层次遍历，用一个队列BFS

但是因为要返回每一层的元素而不是整体返回，我新创建了个队列，来表示当前层的所有子节点，然后再将这个队列复制会判断isEmpty的队列

实际上这样做事复杂了，在每一层时直接取一下当前队列的size然后遍历队列的size个结点就好了(这样做当前层的子节点依然可以加到队列中，代码整洁性比较好)
